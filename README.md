# Telegram Clicker Game template

Here is very base template for tg mini app game creation.
Structure:
1. DB - Postgre SQL sripts for DB instances creation. Seed (mock data) are provided
1. Backend - NodeJS API
1. Frontend - React Telegram Mini App template. TON and Telegram Stars payment inside
1. Admin - managemend DB Entities

Also included:
- Docker
- sh scripts for easy run
